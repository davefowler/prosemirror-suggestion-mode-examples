Prosemirror Suggestion Mode Examples

Prosemirror Suggestion Mode is a plugin for ProseMirror that allows users to make and apply suggestions to the editor similar to Google Docs or Word's suggestion modes.

See the examples:

  • Simple [code] - The most basic example of how to use the library.
  • Basic Prosemirror Example Editor with Markdown [code] - The library connected to the basic prosemirror example editor with a markup menu bar and undo/redo functionality
  • Apply Suggestion [code] - Shows how to use the applySuggestion tool, which is a text based (find/replace) way of making suggestions, ideal for AI's
