Basic Prosemirror Suggestion Mode Example

This example of prosemirror-suggestion-mode with the basic prosemirror example editor using the official example-setup package. It shows how suggestion mode works with and across ProseMirror features such as markup and menuBars.

You can see that edits work across different markup and hover menus for accepting and rejecting each suggestion are displayed on hover over adjacent changes. Three buttons are added to the toolbar:

  • A toggle for enabling/disabling suggestion mode
  • A button to accept all suggestions
  • A button to reject all suggestions
